Economic Valuation Models

Stock-to-Flow Model
PlanB Model
S2F = S / F → ln(P) = a × ln(S2F) + b
S= Current circulating stock
F= Annual flow (new supply - burn)
a= Slope coefficient (≈3.3 for BTC)
b= Intercept constant
PLB Advantage: With negative flow (burn > issuance), S2F → ∞, indicating increasing scarcity premium
Network Value Model
Metcalfe's Law
NV = k × n2 → P = (k × n2) / St
NV= Network value (market cap)
n= Active users/addresses
k= Network coefficient
St= Circulating supply
Proven: Facebook, Ethereum, and Bitcoin valuations follow n² relationship (R² > 0.9)
Token Velocity Model
Fisher Equation
M × V = P × Q → Token Price = (P × Q) / (M × V)
M= Total token supply in circulation
V= Token velocity (turnover rate)
P= Price level of services
Q= Quantity of transactions
PLB Strategy: Staking reduces V (velocity), directly increasing token value per Fisher equation
PLB Integrated Valuation
Combined Model
PPLB = [(k × n2) / St] × [1 + ln(S2F)] × [1 / V]

Theoretical Foundations

S2F
Stock-to-Flow

Developed by pseudonymous analyst PlanB (2019). Models asset value based on scarcity ratio. Successfully predicted Bitcoin's price movements with R² > 0.95.

Reference: "Modeling Bitcoin Value with Scarcity" - PlanB, 2019
n²
Metcalfe's Law

Formulated by Robert Metcalfe (1980). States network value grows proportionally to users squared. Validated across Facebook, Tencent, and major cryptocurrencies.

Reference: "Metcalfe's Law as a Model for Bitcoin's Value" - Clearblocks, 2017
MV
Fisher Equation

Proposed by Irving Fisher (1911). The Equation of Exchange (MV=PQ) is foundational monetary economics. Adapted for crypto token valuation by Chris Burniske.

Reference: "Cryptoasset Valuations" - Chris Burniske, 2017

Academic Basis: PLB tokenomics integrates three peer-reviewed economic frameworks: scarcity-driven valuation (S2F), network effect theory (Metcalfe), and monetary velocity principles (Fisher).
